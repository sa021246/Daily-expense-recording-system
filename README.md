# Expense Tracker 記帳加總器

這是一個用 Python 製作的小型記帳程式，支援：
- 每日總支出加總
- 各類別支出統計
- 繪製支出圖表（使用 matplotlib）

## 執行方式
1. 安裝 Python
2. 安裝套件：`pip install matplotlib`
3. 執行：`python your_file.py`
